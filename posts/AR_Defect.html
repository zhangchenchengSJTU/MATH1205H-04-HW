(缺陷公式). 给定 ses \begin{equation} θ : 0 → K → X → Y → 0. \end{equation} 则有长正合列 \begin{equation} 0 → (-, K) → (-, X) → (-, Y) → K ⊗ \mathrm{Tr}(-) → X ⊗ \mathrm{Tr}(-) → Y ⊗ \mathrm{Tr}(-) → 0. \end{equation}

对投射模的态射 $f$, 取函子 $F(-) = -⊗ f ≃ (D(f), -)$. 此时, 计算 $\mathrm{cok}(F(-))$ 与 $\ker(F(-))$ 无需导出函子. 记 $P_1(-) → P_0(-) → (-)$ 是极小投射表现, 此时

中间是两条短正合列的交换图.

Torsion pair 的遗留问题

($\mathrm{Ext}^1$-垂直条件, 证明). 以下刻画 torsion pair $(𝒯, ℱ)$ 中满足 $\mathrm{Ext}^1$-垂直的对象. 记 $t$ 与 $f$ 是相应的子函子与商函子.

  1. $𝒯 ∩ (^⟂ 𝒯) = 𝒯 ∩ τ ⁻¹ ℱ$.
  2. $𝒯 ∩ (𝒯^⟂) = 𝒯 ∩ t(𝐢𝐧𝐣)$.
  3. $ℱ ∩ (ℱ^⟂) = ℱ ∩ τ 𝒯$.
  4. $ℱ ∩ (^⟂ ℱ) = ℱ ∩ f(𝐩𝐫𝐨𝐣)$.

见此节.

Tilting 与 torsion pair: $(^⟂𝒯) ∩ 𝒯 = 𝐚𝐝𝐝(T)$ 的证明

逼近问题见此节.

(Tilting 对象与 torsion pair, 证明). 对 tilting 对象诱导的 torsion pair $(𝒯, ℱ)$.

  1. 对任意 $M ∈ 𝒯$, 存在满的右 $𝐚𝐝𝐝 (T)$-逼近 $T^n ↠ M$ 使得

    $$ 0 → K → T^n → M → 0 \quad ∈ 𝒯. $$

  2. $(^⟂𝒯) ∩ 𝒯 = 𝐚𝐝𝐝(T)$, 即 $𝒯 ∩ τ ^{-1} ℱ = 𝐚𝐝𝐝(T)$.

    一般来说, 只有 $(^⟂𝒯) ∩ 𝒯 = 𝒯 ∩ τ ^{-1} ℱ$.

  3. $M ∈ 𝒯$ 当且仅当配对 $(T, M) ⊗ T → M$ 是同构.

    一般来说, $M ∈ 𝒯$ 当且仅当配对 $(T, M) ⊗ T → M$ 是满的.

特别地, 存在一列逼近的 ses

$$ 0 → K_{k + 1} → T^{N_n} → K_n → 0\quad (n ≥ 0,\quad K_0 = M). $$

使得有长正合列

$$ θ : \dots → T^{N_2} → T^{N_1} → T^{N_0} → M → 0. $$

使得 $(T, θ)$ 是 $(T, M)$ 的 $\mathrm{End}(T)$-模投射分解.

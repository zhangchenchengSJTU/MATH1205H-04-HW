特征根拾遗

(线性映射的特征根). 选定 $\mathbb F$-线性空间 $V$ 到自身的线性映射

$$\begin{equation} f : V \to V,\quad v \mapsto f(v). \end{equation}$$

称 $\lambda$ 是映射 $f$ 的特征向量, 若存在非零向量 $v$ 使得 $f(v) = \lambda v$.

求导运算

$$\begin{equation} D: C(\mathbb R) \to C(\mathbb R),\quad f \mapsto f' \end{equation}$$

的特征根可以为任何实数 $\lambda \in \mathbb R$. 因为 $D(e^{\lambda x}) = \lambda \cdot e^{\lambda x}$.

假定线性空间维数有限. 若线性映射 $f$ 在某组基下有矩阵表示 $A$, 则 $A$ 的特征值等于 $f$ 的特征值.

相似上三角化的应用

对任意 $A \in \mathbb C^{n \times n}$, 存在可逆矩阵 $P$ 使得 $PQP^{-1}$ 是上三角矩阵.

见 HW15.

给定以 $\mathbb F$ 为系数的多项式 $p(x) = x^n + a_1 x^{n-1} + a_2 x^{n-2} + \cdots + a_n$. 存在 $A \in \mathbb F^{n \times n}$ 使得 $\det (xI - A) = p(x)$.

见 Ex-4, 练习题 A, HW13. 可能需要改变部分角标与符号.

记 $f$ 是整系数的多项式, $p(x) = x^n + a_1 x^{n-1} + a_2 x^{n-2} + \cdots + a_n$ 也是整系数的多项式. 将 $p(x)$ 在 $\mathbb C$ 中分解作一次因子的乘积, 即 $(x-x_1)(x-x_2)\cdots (x-x_n)$. 试证明, $(x-f(x_1))(x-f(x_2))\cdots (f-f(x_n))$ 仍是整系数多项式.

  • 提示: 此题一两句话就能说明白, 请不要想复杂.

Jordan 标准型

熟悉 Jordan 块的结构

(Jordan 块). 将一个以 $\lambda$ 为特征根的 $n$ 阶 Jordan 块为

$$\begin{equation} J_n (\lambda) = \begin{pmatrix} \lambda & 1 & 0 & \cdots & 0 \\ 0 & \lambda & 1 & \cdots & 0 \\ 0 & 0 & \lambda & \cdots & 0 \\ \vdots & \vdots & \vdots & \ddots & 1 \\ 0 & 0 & 0 & \cdots & \lambda \end{pmatrix}. \end{equation}$$

EX-10, HW14 说明任意幂零矩阵相似于某个分块对角矩阵, 各分块形如 $J_s(0)$.

特别注意: 幂零矩阵的 Jordan 型不依赖域的选取.

幂零矩阵的秩与 Jordan 块. 假定 $A$ 是幂零矩阵, 约定 $A^0 = I$. 记 $r_k = r(A^k)$, 可以证明 $\{r_k - r_{k+1}\}$ 是. 以下结论是直接的:

  • $r_0 - r_1$ 是 Jordan 块总数. 因为两者都等于零空间维数;
  • $r_1 - r_2$ 是阶数 $\geq 1$ 的 Jordan 块总数;
  • $r_k - r_{k+1}$ 是阶数 $\geq k$ 的 Jordan 块总数.

因此,

  • $r_0 - 2r_1 + r_2$ 是 $0$ 阶 Jordan 块数量;
  • $r_1 - 2r_2 + r_3$ 是 $1$ 阶 Jordan 块数量;
  • $r_k - 2r_{k+1} + r_{k+2}$ 是 $k$ 阶 Jordan 块数量.

请写出以下矩阵的 Jordan 标准型 (或写出所有的 Jordan 块).

  1. $A$ 是 $3$ 阶矩阵. 满足 $A^2 \neq O$, $A^3 = O$.
  2. $A$ 是 $4$ 阶矩阵. 满足 $r(A) = 2$, $A^2 = O$.
  3. $A$ 是 $5$ 阶矩阵. 满足 $r(A) = 2$, $A^2 \neq O$.
  4. $A$ 是 $6$ 阶矩阵. 满足 $r(A^2) - r(A^3) > r(A^3) - r(A^4) = 1$.

根空间

给定 $n$ 阶矩阵 $A$. 对 $\lambda \in \mathbb F$, 定义 $\mathbb F^n$ 的子空间

$$\begin{equation} V_\lambda := \{v \mid \exists n \in \mathbb N, \ (\lambda I - A)^n v = 0\}. \end{equation}$$

简单地说, $V_\lambda$ 包含一切经 $(\lambda I - A)$ 反复作用后能化为 $0$ 的向量.

直接地, $V_\lambda = 0$ 当且仅当 $\lambda$ 是特征根.

若 $\lambda$ 是特征根, 则对应的特征向量属于 $V_\lambda$; 但 $V_\lambda$ 中的非零向量未必是特征向量, 例如取 $A = \binom{0 \ 1}{0 \ 0}$, 则 $V_0 = \mathbb F^2$, 但 $A$ 的特征向量 (与 $0$) 构成一维空间.

若 $\lambda \neq \mu$, 则线性映射

$$\begin{equation} V_\lambda \to V_\lambda, \quad x \mapsto (\mu I - A)x \end{equation}$$

是双射.

该线性映射的来源与去向维数相同. 因此单, 满, 同构等价. 只需证明单射. 若 $(\mu I - A)x = 0$, 则 $Ax = \mu x$. 由 $V_\lambda$ 的定义, 对足够大的 $k$ 应有

$$\begin{equation} 0 = (\lambda I - A)^k x = (\lambda - \mu)^k x. \end{equation}$$

此时只能有 $x = 0$.

若 $A$ 是复矩阵, 且特征根只有 $\lambda$ 与 $\mu$ ($\lambda \neq \mu$, 不考虑重数). 证明 $\mathbb F^n = V_\lambda \oplus V_\mu$.

  • 提示: 乘积式 $(\lambda I - A)(\mu I - A)$ 是一个幂零矩阵. 由上三角化即可证明.

若 $A$ 是复矩阵, 且特征根只有 $\lambda$ 与 $\mu$ ($\lambda \neq \mu$, 不考虑重数). 此时存在可逆矩阵 $P$ 使得 $PAP^{-1} = \binom{B \ O }{O \ C}$. 其中 $B$ 的特征值只有 $\lambda$, $C$ 的特征值只有 $\mu$.

取 $V_\lambda$ 与 $V_\mu$ 的基, 并将其合并为 $\mathbb F^n$ 的基. 左乘 $A$ 这一线性映射在该组基下的矩阵表示为 $\binom{B \ O }{O \ C}$.

推论: Jordan 标准型

类似的论证给出以下推论.

假定复矩阵 $A$ 的所有特征根为 $\lambda_1, \lambda_2, \ldots, \lambda _m$ (不考虑重数). 则

$$\begin{equation} \mathbb F^n = V_{\lambda _1} \oplus V_{\lambda _2} \oplus \cdots \oplus V_{\lambda_m}. \end{equation}$$

此时存在可逆矩阵 $P$ 使得 $PAP^{-1}$ 是分块对角矩阵, 且第 $k$ 个分块 (记作 $B_k$) 的特征值仅有 $\lambda _k$. 前文表明, $(B_k - \lambda _k I)$ 相似于由若干 $J_t(0)$ 组成的分块对角矩阵, 因此 $B_k$ 相似于由若干 $J_t(\lambda_k)$ 组成的分块对角矩阵. 以上给出 Jordan 型的求解.

练习题

若复矩阵满足 $AB = BA$, 则存在 $v$ 使得 $(\lambda, v)$ 是 $A$ 的特征对, $(\mu, v)$ 是 $B$ 的特征对.

复矩阵 $A$ 有特征根, 取作 $Ax = \lambda x$. 交换性表明 $ABx = BAx = \lambda (Bx)$, 归纳得 $AB^kv = \lambda B^kv$. 记 $V$ 是 $\{x,Bx,B^2,\ldots\}$ 生成的线性空间, 特别地, $V$ 中非零向量都是 $A$ 的特征根. 由于线性映射

$$\begin{equation} V \to V,\quad v \mapsto Bv \end{equation}$$

总有特征根, 取作特征对 $(\mu, v)$. 则 $(\lambda, v)$ 也是 $A$ 的特征对.

证明 $r\binom{A \ B}{O \ D} \geq r\binom{A \ O}{O \ D}$. 取等当且仅当关于 $X$ 与 $Y$ 的方程 $B = AX + YD$ 有解.

($\gets$). 若 $B = AX + YD$, 则 $\binom{A \ B}{O \ D}$ 通过可逆行, 列变换化为 $\binom{A \ O}{O \ D}$.
($\to$). 假定 $A$ 有 $m$ 列. 对 $\binom{A \ B}{O \ D}$ 与 $\binom{A \ O}{O \ D}$ 同步进行 $ref$ 算法, 则两者 $ref$ 的前 $m$ 列相同. 若 $\binom{A \ B}{O \ D}$ 的第 $m + k$ 列能被前面的列线性表示, 取作 $\binom{A \ B}{O \ D} \cdot \binom{x_i}{v_i}$, 则 $\binom{A \ O}{O \ D}$ 的第 $m + k$ 列也能被前面的列线性表示, 取作 $\binom{A \ O}{O \ D} \cdot \binom{0}{v_i}$. 这说明:

  • 若 $\binom{A \ B}{O \ D}$ 的第 $s$ 列非主元列, 则 $\binom{A \ O}{O \ D}$ 的第 $s$ 列也非主元列.

从而 $ref\binom{A \ B}{O \ D}$ 的非零行数大于 $ref\binom{A \ O}{O \ D}$ 的非零行数, 即 $r\binom{A \ B}{O \ D} \geq r\binom{A \ O}{O \ D}$.
取等时, $ref\binom{A \ B}{O \ D}$ 与 $ref\binom{A \ O}{O \ D}$ 主元列位置相同. 记 $X = (x_1 \mid x_2 \mid \cdots \mid x_n)$. 比较 $\binom{A \ O}{O \ D}$ 与 $\binom{A \quad B-AX}{O \quad\quad D\ \ \ \ }$ (抄写前文):

  • 若 $\binom{A \quad B-AX}{O \quad\quad D\ \ \ \ }$ 的第 $m + k$ 列能被前面的列线性表示, 取作 $\binom{A \quad B-AX}{O \quad\quad D\ \ \ \ } \cdot \binom{0}{v_i}$, 则 $\binom{A \ O}{O \ D}$ 的第 $m + k$ 列也能被前面的列线性表示, 取作 $\binom{A \ O}{O \ D} \cdot \binom{0}{v_i}$

从而 $ref\binom{A \quad B-AX}{O \quad\quad D\ \ \ \ } = ref\binom{A \ O}{O \ D}$. 比较行空间, $B-AX$ 的各行能通过 $D$ 各行的线性组合得到. 因此有 $B-AX = YD$.

考虑复矩阵. 假定 $\det(xI - A)$ 与 $\det (xI - D)$ 没有相同的根. 则 $\binom{A \ B}{O \ D}$ 与 $\binom{A \ O}{O \ D}$ 必然相似.

下证明单射:

$$\begin{equation} \varphi : \mathbb F^{m \times n} \to \mathbb F^{m \times n},\quad M \mapsto AM - MD. \end{equation}$$

  • (证明 $\varphi$ 是单射). 考虑三角化 $A = P^{-1}UP$ 与 $D = QWQ^{-1}$. 若 $AM = MD$, 则 $U(PMQ)=(PMQ)W$. 记 $f(x) = \det (xI-A)$, 则 $f(U)^{(n)}(PMQ) = (PMQ)f(W)^{(n)}$. 由假定, $U$ 与 $W$ 的对角元没有相同的元素, 从而 $f(U)^{(n)} = O$ 且 $f(W)^{(n)}$ 可逆. 因此 $PMQ = O$, 即 $M = O$.

线性映射 $\varphi$ 的来源与去向有相同维数, 从而 $\varphi$ 单意味着 $\varphi$ 满. 因此, 关于 $M$ 的方程

$$\begin{equation} \binom{A \ B}{O \ D} \cdot \binom{I \ M}{O \ I} = \binom{I \ M}{O \ I} \cdot \binom{A \ O}{O \ D} \end{equation}$$

有解. 这说明了相似关系.
